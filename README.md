# Сайт KryakOS — официальный лендинг от KryakTeam

> Современная, дерзкая и технологичная презентация операционной системы **KryakOS**, созданной командой энтузиастов **KryakTeam**. Проект полностью выполнен на чистом HTML/CSS/JS без сторонних фреймворков и графических файлов (все визуальные элементы — это CSS и иконки FontAwesome).

---

## 📌 Общая концепция

Одностраничный сайт, посвящённый вымышленной, но детально проработанной ОС **KryakOS**. Сайт передаёт дух high‑tech, свободы и open-source сообщества. Дизайн выполнен в тёмной гамме с неоновыми акцентами, имитацией терминала и glassmorphism эффектами. **Никакие изображения не используются** — вся графика построена на CSS (градиенты, тени, border-radius) и иконочном шрифте.

**Основные цели:**
- Презентовать ключевые возможности KryakOS.
- Познакомить с командой разработчиков.
- Предложить скачать различные редакции системы (демонстрация).
- Создать атмосферу технологического сообщества.

## 🧩 Структура и секции

### 1. Навигация (Navbar)
- Логотип: иконка `fa-bolt` + текст «KryakOS by KryakTeam»
- Якорные ссылки: *Возможности, Скачать, Команда*
- Кнопка «Документация» (вызывает всплывающее уведомление)

### 2. Герой (Hero)
- Бейдж с версией `v3.0 "Phoenix"`
- Заголовок: *Операционная система KryakOS — свобода без границ*
- Терминал (стилизованный `div`) с выводом команд:
  - `uname -a` → KryakOS 6.5.0
  - `kryak --version` → 3.0 (Phoenix)
  - `systemctl status kryak-core` → active (running)
- Две кнопки: «Скачать ISO» и «Узнать больше»

### 3. Блок возможностей (Features)
6 карточек, каждая содержит:
- Иконку (FontAwesome, например `fa-rocket`, `fa-shield-hawk`, `fa-gamepad`)
- Заголовок (Молниеносная, Dev‑first окружение и т.д.)
- Описание технологии или преимущества

**Список возможностей:**
- ⚡ Молниеносная — загрузка за 1.2 секунды
- 💻 Dev‑first окружение — пакетный менеджер KRK
- 🛡️ Безопасность уровня NSA — мандатный контроль доступа
- 🎮 Игровой режим — драйверы для новых GPU
- ☁️ Экосистема KryakCloud — магазин приложений
- 🐉 Открытый код — репозитории на GitHub

### 4. Загрузка (Download)
Выделенный блок с тремя редакциями:
- **Workstation (GNOME)** — для ежедневной работы
- **KryakCore (CLI)** — минималистичная серверная версия
- **Gamer Edition** — оптимизирована для игр

Дополнительная информация: размер 2.8 GB, поддержка архитектур x86_64, ARM64, RISC-V, LTS до 2030 года.

## 🎨 Технические особенности (без изображений)

| Компонент            | Реализация                                                                 |
|----------------------|----------------------------------------------------------------------------|
| Фон                  | Сплошной тёмный `#0a0c12`, градиенты для акцентов                         |
| Карточки             | `background: rgba(18,24,36,0.7)` + `backdrop-filter: blur(2px)`          |
| Терминал             | Чёрный блок с цветными кружками (red/yellow/green) и моноширинным шрифтом  |
| Иконки               | FontAwesome 6 (CDN) — полностью заменяют любые растровые изображения      |
| Анимации             | CSS transition: подъём карточек при hover, изменение цвета обводки        |
| Уведомления (тосты)  | JavaScript + динамическое создание `div` — всплывают снизу                |
| Адаптивность         | Медиазапросы, flexbox- и grid-сетки, резиновые отступы                    |
| Скролл               | Плавная прокрутка к якорям через `scrollIntoView({ behavior: 'smooth' })` |

## 🖱️ Интерактивные элементы (JavaScript)

Все кнопки и ссылки имеют обратную связь без перезагрузки страницы:

- **Кнопки скачивания** — показывают уведомление: «Начинается загрузка KryakOS [Edition] (демо-режим)»
- **Кнопка «Документация»** — уведомление со ссылкой на docs.kryak.org
- **Кнопка «Узнать больше»** — плавный скролл к блоку возможностей + тост
- **Иконки соцсетей** — вместо перехода по мёртвой ссылке выводят информацию о сообществе (например, «GitHub / KryakTeam — репозитории ОС»)
- **Консоль браузера** — при открытии DevTools выводится приветственное сообщение от KryakTeam
